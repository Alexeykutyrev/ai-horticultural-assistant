# AI‑ассистент для промышленного садоводства

**VK‑бот + компьютерное зрение (YOLO) + мультимодальная LLM (Qwen2.5‑VL‑7B) + RAG**

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)

## Обзор

Проект представляет собой интеллектуальную систему поддержки принятия решений для агрономов и научных сотрудников в области садоводства. Система интегрирует:

- **Детерминированное компьютерное зрение** на базе YOLO (Ultralytics) – детекция, сегментация, классификация, оценка ключевых точек.
- **Мультимодальную визуально‑языковую модель** Qwen2.5‑VL‑7B‑Instruct – интерпретация изображений (в том числе размеченных YOLO) и генерация текстовых рекомендаций.
- **Технологию RAG (Retrieval‑Augmented Generation)** – ответы на вопросы с использованием пользовательской базы знаний (PDF, DOCX, TXT).
- **Клиент‑серверную архитектуру** с интерфейсом в **VK Мессенджере** (кроссплатформенный бот) и локальным сервером анализа.

### Возможности

- **Анализ изображений**: подсчёт плодов, цветков, завязей; диагностика болезней листьев (парша, мучнистая роса, ржавчина, пятнистость); распознавание плодовых мушек на световых ловушках; идентификация сортов‑опылителей.
- **Сегментация**: экземпляров (плоды, ветки, плодоножки, цветки, завязи) и семантическая.
- **Оценка структуры кроны**: определение скелета дерева (6 ключевых точек, PCK = 0,83).
- **Агентный анализ (Agentic AI)**: двухэтапный процесс – YOLO размечает, VLM интерпретирует.
- **Сбор датасетов**: автоматическое аннотирование, географическая и метеорологическая привязка, экспорт в COCO, YOLO, Pascal VOC.
- **RAG‑база знаний**: загрузка справочников, поиск релевантных фрагментов, снижение галлюцинаций LLM на 43% (относительное).

## Требования

- **Python 3.10 или выше**
- **ОС**: Windows, Linux, macOS (Apple Silicon или Intel)
- **Оперативная память**: ≥ 16 ГБ, рекомендуется 32 ГБ
- **GPU** (опционально, но рекомендуется для ускорения инференса): NVIDIA с CUDA или Apple M1/M2

## Быстрый старт

### 1. Клонирование репозитория

```bash
git clone https://github.com/Alexeykutyrev/ai-horticultural-assistant.git
cd ai-horticultural-assistant

2. Установка зависимостей

Рекомендуется использовать виртуальное окружение:

bash
python -m venv venv
source venv/bin/activate        # Linux/macOS
venv\Scripts\activate           # Windows

pip install --upgrade pip
pip install -r requirements.txt
3. Скачивание предобученных моделей

Поместите файлы моделей (.pt) в папку models/.
Минимальный набор моделей можно получить по запросу (см. раздел «Данные и модели»).

4. Настройка окружения

Создайте файл .env (не публикуйте его в репозиторий) или экспортируйте переменные:

bash
export VK_GROUP_TOKEN="ваш_токен_группы_VK"
export WEATHER_API_KEY="ключ_OpenWeatherMap"   # опционально

5. Запуск сервера анализа

bash
python server.py
Сервер запустится по адресу http://127.0.0.1:9000. Проверьте работоспособность: curl http://127.0.0.1:9000/health.

6. Запуск VK‑бота

bash
python vk_bot.py
Бот начнёт опрашивать Long Poll API VK. Откройте диалог с вашей группой и отправьте /start.

Структура репозитория

text
ai-horticultural-assistant/
├── server.py                # FastAPI‑сервер компьютерного зрения
├── vk_bot.py                # VK‑бот (клиент)
├── requirements.txt         # зависимости
├── models/                  # папка с моделями YOLO (.pt)
├── user_data/               # пользовательские данные (создаётся автоматически)
│   ├── images/              # загруженные пользователем изображения
│   ├── analyses/            # результаты анализов в JSON
│   ├── documents/           # документы RAG по каждому пользователю
│   ├── rag_index/           # индексы FAISS
│   └── datasets/            # собранные датасеты
└── README.md

Использование бота (основные команды)

/start – главное меню.
📷 Анализ → Анализ изображений → выбор задачи и объекта → отправка фото.
🤖 AI → Agentic AI – интеллектуальный анализ с рекомендациями от VLM.
📊 Данные → Сбор датасета – загрузка фото с автоматической разметкой для формирования обучающей выборки.
📚 База знаний – загрузка PDF/DOCX, поиск с RAG.
⚙️ Настройки – изменение порогов уверенности, IoU, устройства и т.д.
